Konfigurasi database / basis data

Penggunaan basis data / database(akan disebutkan database dibagian selanjutnya) dalam sebuah aplikasi memang sudah menjadi suatu kewajiban, pada pembahasan kali ini saya rasa pasti project skripsi/TAnya teman-teman menggunakan database dan saya asumsikan teman-teman sudah mempunyai gambarkan tentang skruktur database yang teman-teman punya(lihat pada bab 3/4). berikut contoh database dalam pembahasan kali ini tentu tiap-tiap struktur database mempunyai flow yang berbeda tapi jangan khawatir asalkan teman-teman paham dasar dan konsepnya insha Allah mudah semua jalannya.

Contoh database perpustakaan dasar

Yap teman-teman sudah lihat pada pembahasan kali ini kita akan menggunakan database perpustakaan kurang lebih sktrukturnya kaya contoh diatas. Didalam database perpustakaan terdapat 2 tabel yaitu:

  • Tabel users (atau bisa dibilang tabel siswa)
  • Tabel Buku

Selanjutnya untuk definisi tipe data dalam masing-masing tabel akan saya sebutkan (saya sebutkan menggunakan list, btw saya suka merangkumnya dalam list :D)

  • INT tipe data yang digunakan untuk angka yang berukuran normal, Jangkauan nilainya adalah dari -21474883648 hingga 214783647. intinya digunakan untuk nilai yang berbasis angka
  • VARCHAR tipe data yang digunakan untuk nilai yang berbasis string atau campuran antara teks, angka dan simbol, nilai yang diset dalam VARCHAR(nilai) adalah panjangnya, contohnya pada gambar diatas panjangnya mulai dari 222-255
  • TIMESTAMP tipe data yang digunakan untuk nilai yang berbasis dengan tanggal contohnya 2020-06-11 05:12:00, sebenernya mysql memiliki banyak tipe data untuk tanggal seperti DATETIME yang fungsinya hampir mirip. alasan saya menggunakan TIMESTAMP sendiri karena nilai ini tidak akan saya input manual alias akan diinput otomatis sama sistem.

Membuat database pada PHPMyadmin(XAMPP)

Sebelum membahas bagian selanjutnya pastikan teman-teman sudah mengerti alur pembuatan database pada phpmyadmin saya asusmsikan sudah yaa. pada contoh pembahasan kali ini saya akan membuat database perpustakaan, kalau teman-teman punya struktur database sendiri yang hendak dibuat silahkan ikuti saja contoh dibawah ini kurang lebih langkah-langkahnya sama. sekali lagi semoga teman-teman paham sampai disini kalau ada yang hendak ingin ditanyakan silahkan jangan sungkan untuk menuju kontak. selanjutnya buka aplikasi XAMPP(yang sudah diinstall sebelumnya) untuk mengaktifkan server mysql.

Contoh gambar aplikasi XAMPP pertama kali dibuka

Saat aplikasi XAMPP telah terbuka silahkan teman-teman klik start pada bagian mysql dan apache, tunggu sampai berwarna hijau.

Contoh gambar aplikasi XAMPP yang berhasil di start

Perhatian jika teman-teman mendapatkan error saat menjalankan start XAMPP kemungkinan port yang akan digunaakan XAMPP telah digunakan oleh aplikasi lain yang ada pada leptop teman-teman, jangan sungkan untuk menuju kontak dan hubungi sayaa :D

Selanjutnya jika XAMPP sudah aman-aman aja(warna hijau) silahkan buka browser teman-teman dan ketik pada address-bar / url barnya(gunakan mozila/google chrome).

http://localhost/phpmyadmin

Contoh gambar PHPmyadmin localhost kamu

Setelah teman-teman menuju halaman http://localhost/phpmyadmin barusan akan langsung ditampilkan halaman PHPMyadmin seperti contoh gambar diatas. oke langsung saja kita generalisasikan struktrur database perpustakaan pada contoh yang sebelumnya kita buat. Kembali untuk melihatnya

  1. klik pada kata new untuk membuat database baru (lihat kotak merah pada gambar diatas).


Contoh gambar membuat database perpustakaan

  1. Silahkan masukan nama database khusus pada pembahasan kali ini adalah perpustakaan dan klik Create selanjutnya tunggu database akan selesai dibuat.

Contoh gambar membuat tabel users dan jumlah kolomnya

  1. Karena kita baru buat database, maka akan langsung disarankan sama PHPMyadmin untuk segera membuat tabel, langsung saja tanpa menolak buat tabel users lihat contoh diatas dan klik kirim atau go.

Perhatikan untuk bagian Number of Columns nilai awalnya adalah 4 yang mana akan ada 4 kolom dalam tabel users(lihat tabel users). teman-teman juga bisa mengatur jumlah kolom sesuai dengan kemauannya teman-teman khusus saja contoh kali ini menggunakan 4 kolom.



Contoh gambar membuat tabel users

  1. Selanjutnya saat selesai membuat atau mengdeklarasi tabel users, maka teman-teman harus mendefinisikan masing-masing juga tipe data untuk tiap-tiap kolom dan setelah itu klik save untuk menyimpannya. saya membuatnya sesuai dengan rancangan database perpusatkaan kita diawal.

Perhatikan untuk mencentang opsi A_I dan tekan oke jika ada muncul permintaan.


Contoh gambar hasil tabel users yang dibuat

  1. Berhasil tabel users berhasil dibuat berikut contoh gambarnya, sekarang tinggal satu tabel lagi yaitu tabel buku. ayok langsung saja kita buat.



Contoh gambar buat tabel di database perpustakaan

  1. Klik pada nama databasenya teman-teman contoh disini adalah dataabse perpustakaan(lihat gambar diatas). maksudnya adalah kita ingin membuat/menambahkan tabel baru untuk database perpustakaan, sebelumnya tadi jika teman-teman klik new berarti artinya teman-teman akan membuat database baru.


Contoh gambar buat tabel di PHPmyadmin localhost

  1. Silahkan teman-teman masukan nama tabelnya bebas khusus contoh disini adalah buku dan jumlah kolomnya atau Number of Columnsnya adalah 4 sesuai dengan struktur tabel buku yang kita buat (lihat tabel buku). Selanjutnya klik save untuk menyimpan hasilnya


Contoh gambar buat tabel di PHPmyadmin localhost bagian 2

  1. Lanjut juga untuk mendefinisikan tipe datanya untuk tiap-tiap kolomnya, lagi-lagi saya sesuaikan dengan struktur tabel buku yaa :D


Contoh gambar hasil database yang kamu buat di PHPmyadmin localhost

  1. Saat mengklik simpan, maka teman-teman akan melihat hasil tabel yang kita buat untuk saat ini cukup dua(lihat gambar diatas). selebihnya teman-teman juga bisa menambahkan beberapa tabel sesuai dengan kebutuhannya teman-teman.


Sampai disini akhirnya teman-teman sudah berhasil membuat database serta membuat 2 tabel, saya harap semoga teman-teman dapat memahami apa yang saya maksud yaa(maaf yaa klo rada kurang jelas). untuk selanjutnya kita akan menghubungkan database yang baru kita buat perpustakaan dengan aplikasi kita.

Sekali lagi jangan sungkan untuk menuju kontak jika teman-teman merasa kurang puas.

Konfigurasi aplikasi

Untuk memulai ini sebenarnya ada dua cara untuk melakukan konfigurasi pada aplikasi kita, tetapi Alhamdulillah saat ini saya sudah membungkusnya menjadi enviroment-variabel. teman-teman dapat mengaksess file konfigurasinya langsung dengan mengubah filename .env-demo menjadi .env

Perhatian perhatikan didepan mengunakan .(titik)


Setelah berhasil mengubah filenamenya menjadi .env silahkan teman-teman atur konfigurasinya bedasakan lingkungan kerja teman-teman(lingkungan kerja bisa berbeda-beda). saya asumsikan teman-teman menggunakan XAMPP versi Windows kurang lebih konfigurasinya seperti ini (lihat gambar dibawah).

Perhatikan atur konfigurasi DB/database kamu contoh disini menggunakan DB_NAME=perpustkaan / database perpustakaan
