Install kebutuhan awal

Kebutuhan awal ini wajib diikuti yaa agar saat proses pengembangan kedepan tidak ada macet-macet.

Persiapan awal adalah teman-teman diharuskan untuk menginstall aplikasi XAMPP, Sublime text dan Kompor Composer, aplikasi-aplikasi ini diperlukan dalam pengembangan oke langsung saja. lompat dibagian #Install project jika teman-teman sebelumnya sudah pernah menginstall aplikasi yang disebutkan. Pertama silahkan teman-teman download XAMPP yang gunannya untuk menjalankan Webserver atau sederhananya untuk menjalankan aplikasi berbasis PHP. Selanjutnya yang kedua teman-teman silahkan download juga aplikasi Sublime text 3 untuk kebutuhan dalam pengembangan atau penulisan kodingannya dan yang terakhir yang tak kalah penting teman-teman juga harus download Composer. Composer sendiri adalah sebuah package-manager yang dapat membantu kita dalam proses pengembangan aplikasi untuk keperluan bahasa PHP tentunya yaa. yup jangan lama-lama berikut kebutuhan link sudah saya sertakan:

  • Download XAMPP / sudah dilengkapi dengan PHP dan MYSQL

  • Download Sublime text 3 / untuk keperluan koding

  • Download Composer
